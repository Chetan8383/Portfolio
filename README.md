## 📌 About the Project  

This is a responsive **portfolio website** built using modern web technologies. It highlights my **skills, projects, and contact information**, providing an overview of my expertise in **web development**.  

### 🎯 Features  

✔️ **Responsive Design** - Optimized for all screen sizes  
✔️ **Interactive UI** - Smooth animations & transitions  
✔️ **Projects Showcase** - Displays my best work with links

✔️ **SEO Friendly** - Well-structured and optimized for search engines
